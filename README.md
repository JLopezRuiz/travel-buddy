# 🌴 Travel Buddy

Travel Buddy is an AI-powered travel-planning assistant that helps busy travelers plan trips in seconds.

Instead of spending hours researching destinations, users simply enter their **destination and interests**, and an **AI agent powered by Gemini** generates a personalized travel plan including:

* ✈️ Best time of year to visit
* 🏨 Recommended places to stay
* 🎟 Activities and experiences
* 📍 Curated itinerary suggestions

The goal is to make **trip planning effortless and fun**.

---

# 🚀 Features

* **AI Travel Agent**
  Uses Google's Gemini model to generate personalized travel recommendations.

* **Interest-Based Recommendations**
  Travelers can input preferences such as:

  * food
  * nature
  * culture
  * nightlife
  * relaxation

* **Smart Itinerary Suggestions**
  Generates activities and travel timing recommendations tailored to the destination.

* **Modern Frontend**
  Built with React and Next.js for a responsive, fast user experience.

---

# 🧠 How It Works

1. User enters:

   * destination
   * trip length
   * budget level
   * travel interests
3. The application sends a prompt to the **Gemini AI model**
4. The AI agent generates structured travel recommendations
5. The UI displays:

   * ideal travel timing
   * accommodation suggestions
   * activities and itinerary ideas

# ⚙️ Getting Started

## 1. Clone the repo

```bash
git clone https://github.com/JLopezRuiz/travel-buddy.git
cd travel-buddy
```

## 2. Install dependencies

```bash
npm install
```

## 3. Add environment variables

Create a `.env.local` file:

```
GEMINI_API_KEY=your_api_key_here
```

You can obtain an API key from Google AI Studio.

---

## 4. Run the development server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
